# Travel Itinerary & Local Experience Content Maker

This is a client-side web application that generates structured travel plans and monetizable content packs.

## Features
- **Smart Itinerary Generator**: Creates detailed day-by-day plans.
- **Local Experience Explorer**: Suggests hidden gems and cultural tips.
- **Monetization Tools**: Generates affiliate placeholders and digital product outlines.
- **Presets**: Includes built-in templates for Paris, Lisbon, Cancun, Bangkok, and Bali.
- **Theme Support**: Light and Dark modes with brand colors.

## How to Use
1. Open `index.html` in any modern web browser.
2. Select a **Preset** from the top dropdown to see a demo, or fill in the form manually.
3. Click **Generate Plan**.
4. View the result in the **Preview** tab or copy the raw text from the **Markdown Source** tab.

## Tech Stack
- **HTML5**: Semantic structure.
- **CSS3**: Variables, Flexbox, Grid, and responsive design.
- **JavaScript (Vanilla)**: Logic, state management, and template generation.
- **No External Dependencies**: Runs entirely in the browser.

## Customization
To add more "Smart Presets", edit the `PRESETS` object and the `generateItinerary` function in `script.js`.
